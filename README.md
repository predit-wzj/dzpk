# 德州扑克联机版

一个基于 Node.js + WebSocket 的实时多人德州扑克游戏，支持最多 9 人同时游戏。

## 功能特性

### 核心游戏功能
- ✅ 完整的德州扑克规则实现
- ✅ 实时多人对战（最多 9 人）
- ✅ 房间系统（创建/加入房间）
- ✅ 断线重连机制
- ✅ 游戏状态同步
- ✅ 手牌评估与结算

### 高级功能
- 💰 **借贷系统**
  - 玩家间借款（10% 利息）
  - 每局限制一次借款
  - 催收机制（3 次后强制扣款）
  - 自愿还款功能

- 🎮 **游戏管理**
  - 房主踢人功能
  - 中途加入（等待下一局）
  - 游戏结算（排名导出）
  - 筹码重置

- 🎵 **背景音乐**
  - 支持在线音乐链接
  - 支持本地音频文件
  - 循环播放
  - 手动控制开关

- 🎨 **界面优化**
  - 响应式设计
  - 清晰的牌面显示
  - 实时游戏日志
  - 玩家状态标识（离线/等待/全押）

## 技术栈

- **后端**: Node.js + WebSocket (ws 8.18.0)
- **前端**: 原生 HTML/CSS/JavaScript
- **音频**: HTML5 Audio API + Web Audio API

## 快速开始

### 环境要求

- Node.js 20.x 或更高版本
- npm 包管理器
- （可选）PM2 进程管理器

### 安装部署

#### 方式一：自动部署（推荐）

```bash
# 克隆项目
git clone <repository-url>
cd dzpk

# 运行初始化脚本
chmod +x init.sh
./init.sh
```

#### 方式二：手动部署

```bash
# 1. 安装依赖
npm install --production

# 2. 配置背景音乐（可选）
# 编辑 conf.env 文件，设置 BGM_URL

# 3. 启动服务
node server.js

# 或使用 PM2
pm2 start server.js --name dzpk
```

### 访问游戏

启动成功后，在浏览器访问：
```
http://localhost:3000
```

或局域网内其他设备访问：
```
http://服务器IP:3000
```

## 配置说明

### conf.env 配置文件

```bash
# 背景音乐直链（支持 .mp3 格式）
BGM_URL=https://music.163.com/song/media/outer/url?id=歌曲ID.mp3

# 或使用本地文件
BGM_URL=/bgm.mp3
```

**获取网易云音乐直链：**
1. 打开网易云音乐网页版
2. 找到想要的歌曲，查看 URL 中的歌曲 ID
3. 使用格式：`https://music.163.com/song/media/outer/url?id=歌曲ID.mp3`

## 游戏说明

### 创建房间

1. 输入玩家昵称
2. 设置初始筹码（默认 1000）
3. 设置小盲注（默认 10）
4. 点击"创建房间"
5. 分享房间号给其他玩家

### 加入房间

1. 输入玩家昵称
2. 输入房间号
3. 点击"加入房间"

### 游戏操作

- **弃牌 (Fold)**: 放弃本局
- **过牌 (Check)**: 不下注继续游戏
- **跟注 (Call)**: 跟上当前下注
- **加注 (Raise)**: 增加下注金额
- **全押 (All-In)**: 押上所有筹码

### 借贷系统

1. 点击其他玩家座位上的"借钱"按钮
2. 输入借款金额（必须是 10 的整数倍）
3. 等待对方同意
4. 借款成功后需支付 10% 利息
5. 可随时主动还款
6. 出借人可催收（3 次后强制扣款）

### 房主功能

- **踢出玩家**: 点击玩家座位上的"踢出"按钮
- **游戏结算**: 点击右上角"游戏结算"按钮
  - 查看所有玩家排名（考虑负债）
  - 结果自动保存到 `res.log` 文件
  - 游戏重置，回到等待室

## 常见问题

### 1. 背景音乐无法播放？

- 检查 `conf.env` 中的 `BGM_URL` 是否正确
- 确认音乐链接可访问
- 尝试使用本地音频文件
- 检查浏览器控制台是否有错误信息

### 2. 游戏卡住不动？

- 检查是否有玩家离线
- 离线玩家会自动弃牌
- 房主可以踢出离线玩家

### 3. 无法加入房间？

- 确认房间号正确
- 检查房间是否已满（最多 9 人，不包括被踢出的玩家）
- 确认网络连接正常

### 4. 断线重连失败？

- 清除浏览器缓存
- 刷新页面重新加入房间
- 检查服务器是否正常运行
